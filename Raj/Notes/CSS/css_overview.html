Cascading, `style` Attribute, `style` Container, External CSS Files **Theory:** - **Cascading:** Styles can override one another based on source (inline > internal > external). - **`style` Attribute:** Inline styling directly on elements. - **Internal CSS:** Using ` ``` - **External:** ```html ``` **Explanation:** Order of specificity matters; inline styles override others, internal can override external, etc. --- ## 7. CSS Properties ### a. Color Properties **Example:** ```css body { color: #222; background-color: #f7f7fb; } ``` **Explanation:** Sets text and background colors. ### b. Font Properties **Example:** ```css p { font-family: Verdana, sans-serif; font-size: 16px; } ``` **Explanation:** Controls font family and size. ### c. Line-height Property **Example:** ```css p { line-height: 1.5; } ``` **Explanation:** Spacing between lines of text. ### d. Text Properties **Example:** ```css h2 { text-align: right; text-decoration: underline; } ``` **Explanation:** Aligns and decorates text. ### e. Border Properties **Example:** ```css div { border: 1px solid #000; } ``` **Explanation:** Applies a border to elements. ### f. Element Box (Box Model), Padding, Margin **Theory:** The CSS box model describes element layout: - **Content**: The text/image/element itself. - **Padding**: Space inside border, around content. - **Border**: Surrounds the padding (and content). - **Margin**: Space outside the border. **Example:** ```css .card { padding: 20px; border: 2px solid #eee; margin: 12px; } ``` **Explanation:** - `padding`: Space inside the card. - `border`: Line around the card. - `margin`: Space outside the card. --- ## 8. Hosting a Website and GIT **Theory:** - **Hosting:** Upload your site files (HTML, CSS, etc.) to a server so anyone can see it online. - **GIT:** A version control system to track code changes and collaborate. **Example Workflow:** 1. Create a GitHub repository. 2. Push your website files. 3. Deploy using services like GitHub Pages or Netlify. **Explanation:** GIT helps you manage code versions, and hosting makes your website live.
