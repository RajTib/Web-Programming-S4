Web Programming Foundations

The Web Programming Foundations course introduces learners to the core technologies behind modern websites. It builds a strong base in HTML, CSS, and fundamental web concepts, preparing students for advanced frontend and backend development.

Topics Covered

  • HTML5 Semantic Elements
  • CSS Basics and Layouts
  • Responsive Web Design
  • Web Accessibility
  • Introduction to JavaScript

Course Modules

Module No. Module Name Duration
1 HTML Fundamentals 2 Weeks
2 CSS & Layout Techniques 3 Weeks
3 Responsive Design 2 Weeks
4 Intro to JavaScript 3 Weeks
